# Следующие шаги для добавления освещения

Сейчас у вас работает базовая визуализация октаэдра без освещения.

Чтобы добавить освещение Блинн-Фонга обратно, нужно:

1. Восстановить полные структуры SceneUniforms и ModelUniforms с правильным выравниванием
2. Обновить шейдеры для поддержки освещения
3. Убедиться что все поля структур правильно выровнены по std140

## Проблема которая была

Структуры в C++ и GLSL должны иметь одинаковое выравнивание памяти.
Для std140 layout:
- vec3 занимает 16 байт (как vec4)
- mat4 занимает 64 байта
- Каждое поле должно быть выровнено по своему размеру

## Решение

Использовать padding (_pad0, _pad1 и т.д.) чтобы обеспечить правильное выравнивание.
